# ECA 56 – MATLAB Wireless Communication & 5G Laboratory Experiments

## 📖 Overview

This repository contains MATLAB implementations of the **ECA 56 – Wireless Communication and 5G Networks Laboratory** experiments. The experiments cover the fundamentals of multiple access techniques, **5G New Radio (NR), MIMO systems, beamforming, IoT, V2X communication, and Artificial Intelligence** applications in modern wireless networks.

These programs are intended for **academic learning, simulation, and performance analysis** using MATLAB.

---

# 🧪 Experiments

**Objective:**  
Simulate Frequency Division Multiple Access (FDMA) by allocating separate frequency bands to multiple users.

### Topics Covered

# 🎯 Learning Outcomes

After completing these experiments, students will be able to:

- Understand multiple access techniques (FDMA, TDMA, OFDMA).
- Analyze propagation characteristics of different 5G frequency bands.
- Evaluate MIMO and Massive MIMO performance.
- Study beamforming and antenna steering techniques.
- Analyze 5G NR frame structures and numerology.
- Simulate IoT and V2X communication scenarios.
- Compare NB-IoT and LTE-M technologies.
- Evaluate throughput and BER performance.
- Apply AI techniques for wireless network traffic prediction.
